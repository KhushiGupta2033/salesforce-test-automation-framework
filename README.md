# Salesforce-test-automation-framework
A scalable test automation framework for Salesforce built with Java, Selenium, and TestNG, following industry best practices like POM and modular architecture.

# Tech Stack
- Java  
- Selenium WebDriver  
- TestNG  
- Maven
- Jenkins

# Features
- Page Object Model (POM)
- Modular framework design
- Reusable utilities
- Data-driven testing
- Easy to scale and maintain
